1.数据库服务器
  当前主流的数据库是——关系型数据库(RDBMS)。分为两部分：
  RDBMS服务器软件：负责存储和管理数据(对于人来看都是乱码)
  RDBMS客户端软件：负责向服务器发起增删改查命令(显示出人能读懂的操作结果)
   		
 
数据库服务器中的数据结构：
